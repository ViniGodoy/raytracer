Java Raytracer
==============

Project description
-------------------

A Simple Java raytracer written in Java 2D.

The code is based in [Ray Tracing from the Ground Up book][1] and in [this excelent tutorial of flipcode][2].
With some refactorings, corrections and ideas of my own.

Change History
--------------
